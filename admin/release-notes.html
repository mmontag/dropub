Version 1.1

  • Added the ability to set "group" and "others" file mode (permissions) on a file before upload. This solves the issue with images dragged from Google Chrome, which do not set either group nor others mode bits.

    This feature is enabled by default, but will only set read bits for group and other. You will have to manually enable write and/or execution bits in the "General" tab, in the Preferences window.

  • Minor UI fixes in the Preferences window.
